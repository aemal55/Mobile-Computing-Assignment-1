Personas of the App

Anyone could use this translation app since many users are interested in knowing which language they should use to translate the text from, such as from Russian to English or English to Russian. Users can also interact with the app by speaking into it, which will enable them to easily understand the text in speech form if they choose to do so; if not, they can use the text as is to understand what they need to know.

The app also uses history; all the converted text's history can be saved if the user so chooses, and if the user so chooses to delete either the "Favorites" or "History," they can do so by using the edit button. Users can use the app to mark and save their translated text so that the next time they come, they can view the selected ones for reminder.
